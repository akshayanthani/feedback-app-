# Feedback App (C + HTML Portfolio Project)

A simple feedback collection web app using a beautiful HTML frontend and C CGI backend.

## Features
- Clean responsive HTML/CSS feedback form
- Backend logic written in C (CGI script)
- Stores user feedback in a text file
